# Sharing
- fetch (скачивание объектов и ссылок из другого репозитория)

    - -a/--append (добавить имена ссылок и объектов к полученным ссылкам)

    -  --deepen=< depth > (как depth, только не с вершины)
    - --shallow-since=< date > (Углубите или сократите историю мелкого(?) хранилища, чтобы включить все достижимые коммиты после < date >)
    - --shallow-exclude=< revision > (Углубление или сокращение истории неглубокого хранилища, позволяющее исключить указанную ветку или тега)
    - --unshallow (преобразует мелкий репозиторий в полный, удалив все ограничения, наложенные мелкими репозиториями.)
    - [ ] --update-shallow
    - --dry-run (показывает, что будет сделано, без выполнения команды)
    - [ ] -f/--force
    - -k/--keep (хранить загруженный пакет)
    - [ ] --multiple (Разрешите указывать несколько аргументов < repository > и < group >, но не < refspec >)
    - -p/--prune (Перед извлечением удаляет ссылки удаленного слежения)
    - -P/--prune-tags (Перед извлечением удаляет все локальные теги, которые больше не существуют на удаленном компьютере)
    - -n/--no-tags (отключает автоматическую загрузку тега)
    - [ ] --refmap=< refspec > (сопоставить ссылки refs с удаленно отслеживаемыми ветками)
    - -t/--tags (Извлечение всех тегов из удаленной ветки)
